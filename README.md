# SQL_Music_Store_Analysis-
Digital Music Store data analysis using SQL

## Project Overview
This project analyzes a music store database using SQL to extract insights on customer behavior, sales trends, and popular genres.

## Database and Tools
* Postgre SQL
* PgAdmin4


## Key Analysis
- Top customers by spending
- Most popular music genres
- Revenue by country
- Best-selling artists

## Insights
- USA generates highest revenue
- Rock genre dominates sales
- Top customers contribute major revenue

## Project Files
- schema.sql → Database structure
- queries.sql → SQL queries
- insights.md → Key findings

## What I Learned
- Writing complex SQL queries
- Using joins, aggregations, subqueries
- Data-driven decision making
